Http Methods Path Description
GET /users Display all users.
GET /users/:id Display a single user
GET /items Display all products
GET /items/:id Display single product
GET /users/:id/carts Display Cart
PUT /users/:id Update single user record
PUT /users/:id/cart/:id Update cart item
PUT /items/:id Update a single products
PATCH /users/:id To modify a user's record.
PATCH /items/:id Update products
PATCH /users/:id/cart/:id To modify a cart item
POST /register Add new user to table
POST /login Create user token
POST /user/:id/cart Add to cart
POST /items Add product
DELETE /users/:id To delete a single user.
DELETE /items/:id Delete a product.
DELETE /items/:id/cart/:id Delete a product from cart
DELETE /items/:id/cart Delete cart
